# Diabetes Prediction & Analysis

This project explores a large dataset to understand the key factors influencing diabetes and builds a machine learning model for prediction.

## 📊 Project Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of key trends
- Machine learning model for prediction

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## 📁 Dataset
- CDC Diabetes Health Indicators Dataset (UCI)
